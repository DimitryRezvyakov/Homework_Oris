====Как запускать===================================================================================================================
1. Скачать /src

Далее либо:-------------------------------------------------------------------------------------------------------------------------

	1.1 Создать БД в PostgreSQL и использовать /src/init.sql для заполнения БД
	1.2 Запустить приложение в IDE, перед этим настроив в settings.json:
									1)строку подключения к БД
									2)порт и домен приложения (по умолчанию http://localhost:8888/)
	1.3 Зайти на сайт в браузере. http://домен:порт/

Либо:-------------------------------------------------------------------------------------------------------------------------------

	2.1 Скачать Docker Desktop и настроить под Linux
	2.2 Поменять домен приложения на +, а Host в строке подключения к БД на postgres, если меняли порт, то в Dockerfile поменять 		    Expose 8888 на Expose (ваш порт)
	2.2 Ввести команду docker built -t (НАЗВАНИЕ - по умолчанию mini-http-server) -f Presentation/Dockerfile .
	2.3 Если указали другое название, то в docker-compose.yml поменять app -> image на ваше название
	2.4 Если меняли порт, то в docker-compose.yml поменять все порты (8888) на ваш порт
	2.5 Поменять все данные БД на свои в docker-compose.yml
	2.5 Ввести команду docker-compose up -d (Если далее нужно будет привести БД к первоначальному виду, то перед запуском писать 							docker-compose down -v)
	2.6 Перейти на сайт. Чтобы остановить приложение нужно написать команду docker-compose down
